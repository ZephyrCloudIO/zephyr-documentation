# Generate an API Token

If you need to create an API token (for CI/CD or any other automated tasks). The steps are simple:

<Steps>
### Click on your avatar
In the upper right you should see your users avatar with a drop down

### Select My Profile
One of the options in this menu should be "My Profile"

### Selet the settings tab
Once on the profile page selec the settings tab from the tab options

### Select API Tokens from the left navigation
On the left hand navigation there should be an options for "API Token" select this option

### Click Generate Token Button
Fill out the required information about the token you're about to generate

![token screen](../public/token-screen.png)

### Click Generate

Copy the token for usage where you need to use it

</Steps>
