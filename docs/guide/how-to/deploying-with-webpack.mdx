# Deploying with webpack
You can deploy an existing webpack-bundled application to Zephyr Cloud.
Support for webpack is facilitated via the `zephyr-webpack-plugin` package.

---

<Steps>
### Install the webpack plugin
<PackageManagerTabs command="install zephyr-webpack-plugin --save" />

### Add the plugin to your webpack configuration
Add the following import to your webpack configuration.

<Tabs values={[{label: 'ES6 Module'}, {label: 'CommonJS'}]}>
    <Tab>
    ```js filename="webpack.config.mjs"
    import { withZephyr } from 'zephyr-webpack-plugin';
    ```
    </Tab>
    <Tab>
    ```js filename="webpack.config.js"
    const { withZephyr } = require('zephyr-webpack-plugin');
    ```
    </Tab>
</Tabs>

Export a valid webpack configuration from `webpack.config.js` wrapped in a call to a function returned by `withZephyr()`.

<Tabs values={[{label: 'ES6 Module'}, {label: 'CommonJS'}]}>
<Tab>
```js filename="webpack.config.mjs"
export default withZephyr()({
    // webpack configuration here
});
```
</Tab>
<Tab>
```js filename="webpack.config.js"
module.exports = withZephyr()({
    // webpack configuration here
});
```
</Tab>
</Tabs>
</Steps>

When you next run webpack, a snapshot of your application is deployed to Zephyr Cloud.

## Using webpack with Nx Composable Plugins
If your project uses webpack via Nx's Composable Plugins feature, you can enable Zephyr Cloud by adding a call to `withZephyr()` as the last argument to `composePlugins()`.

<Tabs values={[{label: 'ES6 Module'}, {label: 'CommonJS'}]}>
<Tab>
```js filename="webpack.config.mjs"
export default composePlugins(
    withNx(),
    // Additional webpack plugins
    withZephyr()
);
```
</Tab>
<Tab>
```js filename="webpack.config.js"
module.exports = composePlugins(
    withNx(),
    // Additional webpack plugins
    withZephyr()
);
```
</Tab>
</Tabs>
