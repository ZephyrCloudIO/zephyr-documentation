# Using the browser extension
The Zephyr Mission Control browser extension provides provides a quick way to see your applications live, toggle between tagged deployments, and view contextual information about your application.

:::note
Official support for the extension is currently limited to Google Chrome.
:::

## Google Chrome

### Installing the extension

<Steps>
### Open the extension in the Chrome Web Store

<Button
    text="Open the Chrome Web Store"
    href="https://chromewebstore.google.com/detail/zephyr-mission-control/liflhldchhinbaeplljlplhnbkdidedn"
    size="medium"
    external="true"
/>

### Add the extension to Chrome
Click the **Add to Chrome** button, followed by **Add extension**.

### Pin the extension beside the address bar
Click the **Extensions** button to the right of the address bar.
To the right of **Zephyr Mission Control**, click the **Pin** button.

### Link the extension to Zephyr Cloud
Right click the **Zephyr Mission Control** icon to the right of the address bar.
Click **Open side panel**.
Click the **Login** button and authenticate with Zephyr Cloud.

</Steps>

### Viewing an application

<Steps>
### Open the side panel
Click the **Zephyr Mission Control** icon to the right of the address bar.
### Select an application
In the **Application** dropdown menu, select the application you'd like to view.
</Steps>

### Enabling live reload
You can enable Live Reload while working on your application.
When enabled, your browser will automatically load the latest deployment of your application.

<Steps>
### View an application using the extension
Follow the [Viewing an Application](#viewing-an-application) procedure.

### Enable live reload in the side panel
Under the **Tags** dropdown menu, enable the **Live reload** checkbox. 
</Steps>
