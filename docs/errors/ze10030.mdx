This error occurs when Zephyr fails to receive a response from the server when fetching the application configuration.

## Debugging the error

<Steps>

### Check your network connection

Ensure your network connection is stable and you can reach Zephyr's servers.

### Verify authentication

Make sure you're properly authenticated by logging in from the [dashboard](https://app.zephyr-cloud.io) or [Chrome extension](https://chromewebstore.google.com/detail/zephyr-mission-control/liflhldchhinbaeplljlplhnbkdidedn).

### Retry the build

This error can sometimes be transient. Try running the build again:

```bash
npm run build
```

</Steps>
